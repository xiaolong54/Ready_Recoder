## 🎯 使用方法

### 操作流程
1. 点击"房间管理" → "名字/ID 添加"
2. 选择平台 (抖音/哔哩哔哩/斗鱼)
3. 输入搜索内容 (主播名称/ID/URL)
4. 点击"搜索"按钮
5. 查看搜索结果,包含:
   - 主播名称
   - 平台ID
   - 粉丝数 (友好格式)
   - 开播状态 (带符号)
6. 选中目标主播
7. 点击"添加选中"按钮,完成添加

## 🔧 技术细节

### 数据流
```
用户输入 → GUI → RoomManager → TargetResolver
                                    ↓
                          DouyinAPIClient.search_user (获取粉丝数)
                                    ↓
                          PlatformParser.get_live_status (异步查询开播状态)
                                    ↓
                          合并结果 → GUI显示
```

### 性能优化
- 使用 ThreadPoolExecutor 并发查询开播状态
- 查询超时 5 秒,避免长时间等待
- 查询失败不影响整体流程,显示"未知"状态

### 兼容性
- 保持现有架构不变
- 向后兼容原有配置
- 不影响其他功能模块

## 📝 注意事项

1. **开播状态查询**: 需要网络请求,首次搜索时会有延迟(约5-10秒)
2. **粉丝数显示**: 仅支持抖音平台名称搜索获取,B站和斗鱼暂未实现
3. **自动录制**: 保留了开播自动录制选项,默认开启
4. **ID查询**: 输入纯数字ID时,粉丝数显示为0(未查询用户信息)

3. **头像显示**: 在搜索结果中显示主播头像
