## 三、使用说明

### 3.1 推荐使用方式

由于抖音API限制,推荐使用以下方式添加直播间:

#### 方式1: 通过房间ID添加
1. 打开抖音APP或网页版
2. 进入目标主播的直播间
3. 复制房间ID (URL中的数字部分,如 `live.douyin.com/83492054`)
4. 在本程序中点击"房间管理" → "名字/ID 添加"
5. 输入房间ID,点击搜索
6. 点击"添加选中"按钮完成添加

#### 方式2: 通过直播URL添加
1. 复制完整的直播间链接
   ```
   https://live.douyin.com/83492054
   ```
2. 粘贴到搜索框
3. 点击搜索,系统会自动解析
4. 点击"添加选中"按钮完成添加

### 3.2 操作快捷键

- **Enter**: 搜索
- **双击列表项**: 直接添加选中的直播间

## 五、已知限制

1. **抖音API验证限制**
   - 名称搜索需要验证码或登录状态
   - 无法直接通过关键词搜索主播
   - **解决方案**: 使用房间ID或完整URL

2. **开播状态查询**
   - 某些房间ID可能返回错误
   - 查询超时设为5秒,避免长时间等待
   - **解决方案**: 失败时显示"未知"状态

3. **粉丝数获取**
   - ID查询时无法获取粉丝数(需要名称搜索)
   - **解决方案**: ID查询显示"-"符号
